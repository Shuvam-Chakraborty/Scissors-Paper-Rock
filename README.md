# ✂️ Scissors Paper Rock 

A terminal-based Scissors Paper Rock game against the computer, played over **10 rounds**.

---

## ▶️ How to Play

```bash
python Scissors-Paper-Rock.py
```

Enter your choice each turn:

| Input | Move |
|-------|------|
| `S` | Scissors |
| `P` | Paper |
| `R` | Rock |

---

## 🎮 Features

- 10-round match against a randomized computer opponent
- Invalid input is rejected without wasting a turn
- Live result shown after each round
- Final scoreboard with wins, losses, and draws

---

## 📈 Sample Output

```
Turn 1 :
Enter your choice : R
You have entered Rock.
Computer has entered Scissors.
You won.

Score:
You: 6  Computer: 3  Draw: 1

Final result: You have won. Congratulations!!!
```

---

## 📦 Requirements

Python 3.x — uses only the built-in `random` module, no installs needed.
